# KK 4-day 커스텀 (nSuns 4-day 변형) — 설계 초안 v0 (과학 검증 전)

> 사용자 지시(2026-07-11): ①데드 매주 고중량 불가 → heavy/volume 2주 교차 ②5일 피로 과다·허리 부상 발생 → nSuns 4-day 골격 ③T1/T2 세트 과다 → 부위당 주 ~20세트 목표, T1 후반부 무게변화 최소·세트 감축·마지막 AMRAP 유지, T2는 웜업 제외 5세트 고정 rep·마지막 세트 경량 ④OHP T1 승격(벤치 volume은 T2로) ⑤월요일 등 day 추가(등 메인은 1+ 없이 6~9렙, 9렙 도달 시 증량) ⑥일요일 크로스핏(외부 기록).

| 요일 | 메인 | 보조 T2 | 보조 | abs |
|---|---|---|---|---|
| 월 | T-bar 로우 (DP 6~9) | 풀업 (DP 6~10) | 백 익스텐션 | O |
| 화 | 데드 (주1=volume, 주2=heavy 교차) | 프론트 스쿼트 | 힙 쓰러스트 | - |

부상 고려로 **week0 = 데드 volume, week1 = 데드 heavy** (첫 주를 가볍게 시작).

## 세트 스킴
**T1 표준 (스쿼트·벤치·OHP·데드heavy) — 6세트, nsunsTopSet 규칙:**
`65%×5 → 80%×3 → 95%×1+ (topSet) → 80%×3 → 80%×3 → 80%×3+ (backoff AMRAP)`
- 백오프 3세트 동일 무게(원판 교체 0회), 마지막 AMRAP 유지. 증량: 벤치·OHP +2.5 / 스쿼트·데드 +5.
- (기존 9세트 → 6세트)

**데드 volume 주 — 5세트, 규칙·AMRAP 없음:** `60%×5 → 70%×5 ×4`

**T2 독립 리프트 (프론트·스모·CGBP) — 5세트 고정 rep + 마지막 경량, t2LastSet 규칙:**
- 프론트: `55%×8 ×4 → 45%×10` (+5)
- 스모: `60%×6 ×4 → 50%×8` (+2.5)
- CGBP: `60%×8 ×4 → 50%×10` (+2.5)
- 판정 = 마지막(경량) 세트 완수 → +스텝. 주1회 노출이므로 진행속도 = nSuns 표준(상체 +2.5/주, 프론트 +5/주)과 동일.

**벤치 T2 (수, volume 노출) — 규칙 없음(벤치 TM은 토 topSet만 판정, 불변식):** `65%×8 ×4 → 55%×10`

**등 메인(T-bar)·풀업 — tracked + doubleProgression (기존 규칙 재사용, 신규 규칙 불필요):**
- T-bar: 4세트, repMin 6·repMax 9·step 5 → **마지막 세트 9렙 도달 시 +5kg, rep 리셋** (사용자 지시 "9 찍으면 증량" 정확 부합)
- 풀업: 4세트, repMin 6·repMax 10·step 2.5 (중량조끼/벨트, 맨몸은 0kg 입력)

**악세사리 (전부 DP 3세트):** 백익스텐션 10~15 / 힙쓰러스트 8~12 / 레터럴 12~20 / 카프 12~20(기존) / 컬 8~12(기존) / abs(행잉 레그레이즈) 10~15 ×월·수

## 앱 반영 방식 (도메인 무수정)
- 새 프로그램 JSON `programs/kk-4day.json` (2주 weeks 배열 — 엔진·검증기·커서 멀티위크 지원 확인 완료). 기존 TM(벤치·OHP·스쿼트·데드·프론트·스모·CGBP) 전부 승계 — 재온보딩 불필요, 전환만.
- T-bar·풀업 등 신규 tracked 리프트는 needsInit 자동 부트스트랩(첫 세션에 무게 직접 입력).
- exerciseLibrary에 신규 6종 추가(tbarRow·pullup·backExtension·hipThrust·lateralRaise·legRaise — 데이터 추가만, hinge 플래그 없음 유지로 기존 테스트 불변).
- **5kg 단위 요청**: 플레이트 설정에서 1.25kg 제거한 사용자 설정(src/lib/plateConfig.ts)을 주입 → 전 무게 5kg 단위 표시(도메인 무수정, stepOf=5). 트레이드오프: T2·스테퍼도 5kg 단위가 됨(보고서에 명시).
- ProgramLibrary에 "내장 프로그램" 섹션 추가(번들 JSON을 버튼 한 번으로 라이브러리 등록 — 폰에서 URL 타이핑 불필요).

## 미해결(과학 검증 대상)
후반 백오프 80% 3×3의 강도-볼륨 적정성 / T2 % 상향(스모 60·CGBP 60·프론트 55)의 근거 / 부위별 세트수가 근거 범위(10~20)에 드는지 / 데드 격주 heavy의 근거 / 급성 허리부상 직후 복귀 프로토콜과의 충돌 여부.
